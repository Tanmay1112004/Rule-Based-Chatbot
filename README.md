# 💬 NLTK + Streamlit Chatbot

A lightweight, rule-based chatbot using NLTK's `Chat` with a Streamlit frontend. Built with love by Tanmay.

## Quickstart

```bash
# 1) Create & activate a venv (recommended)
python -m venv .venv
# Windows:
.venv\Scripts\activate
# macOS/Linux:
source .venv/bin/activate

# 2) Install deps
pip install -r requirements.txt

# 3) Run
streamlit run app.py
